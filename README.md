# 🎴 Generador de Tarjetas

Generador de tarjetas personalizadas con foto y mensaje, listo para imprimir.

## ✨ Funciones

- Subir foto personalizada
- Gestionar múltiples mensajes (agregar, editar, eliminar)
- 20 colores de marco exterior + slider de tamaño
- Slider de tamaño de foto
- 10 colores de fondo
- 8 colores de texto
- 7 tipografías
- 4 layouts (texto arriba/abajo, foto izq/der)
- Alineación de texto (centrado / izquierda)
- Exportar PNG 3× (alta resolución para impresión)

## 🚀 Uso

Abrir `index.html` directamente en el navegador, o acceder vía GitHub Pages.

## 📁 Estructura

```
generador-tarjetas/
├── index.html   ← estructura HTML
├── style.css    ← estilos
├── app.js       ← lógica JavaScript
└── README.md
```

## 🌐 GitHub Pages

Una vez subido al repositorio, activar GitHub Pages desde:
`Settings → Pages → Branch: main → / (root)`

La URL será: `https://TU-USUARIO.github.io/generador-tarjetas/`
